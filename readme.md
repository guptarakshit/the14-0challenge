# 🏏 14-0: The IPL Challenge

Can you build an IPL team good enough to go undefeated?

14-0 is a browser-based cricket drafting and management game where players draft IPL legends and stars from random franchises and seasons, then attempt to complete a perfect 14-0 IPL league campaign against projected 2026 IPL teams.

## Features

- Randomized IPL Draft System
- 11 Round Team Building
- 5-Player Draft Choices
- Knowledge Mode
- Team Chemistry System
- Season Simulation Engine
- 14-0 Leaderboard
- Hall of Fame
- Match Progress Tracker

## How To Play

1. Enter your name and team name
2. Complete the 11-round draft
3. Select your captain
4. Simulate the season
5. Try to finish 14-0
